## What's New

### Dashboard Analytics Widgets
Four new widgets with glassmorphism styling and smooth entrance animations.

**Cache Growth Trend** - Sparkline visualization showing cache growth over time. Displays current usage, daily growth rate, and estimates how many days until your cache is full based on recent patterns.

**Peak Usage Hours** - A 24-hour heatmap showing download activity throughout the day. Highlights both the peak hour and current hour, with hover tooltips showing exact download counts and bytes served.

**Client Efficiency Ranks** - Leaderboard of your top 5 clients ranked by cache hit ratio. Shows bandwidth saved per client with gold/silver/bronze indicators for top performers.

**Game Cache Performance** - Lists top games by bandwidth saved, showing hit ratios and download counts for each title. Helps identify which games benefit most from caching.

### Guest Mode Lock
Admins can now lock guest mode entirely to prevent new guest sessions from being created.

- New lock/unlock toggle in the User tab
- Existing sessions remain active until expiration
- Clear visual indicator shows current lock state
- Takes effect immediately when toggled

### Dashboard Card Animations
Dashboard cards now animate in with a staggered entrance effect. Each card fades in and slides up with a subtle blur transition, creating a polished loading experience.
